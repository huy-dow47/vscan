# vscan
This is a vulner scan script made in python with new options
## Options
```
    -u, --url: Specify the URL of the website to scan for vulnerabilities.
    -s, --silent: Run the script in silent mode, suppressing unnecessary output.
    -c, --checkall: Perform a comprehensive vulnerability check, including all available tests.
    --ssl: Enable SSL verification for HTTPS connections.
    --cookies: Include cookies in the requests made during vulnerability scanning.
    --content: Display the response content of each request made during scanning.
    --proxy: Specify a proxy server to use for the requests.
    --timeout: Set the timeout value (in seconds) for the requests made during scanning.
    --redirects: Follow redirects during scanning and display information about them.
```
## Install
Launch the init.bat file
